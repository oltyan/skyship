SkyShip leverages a number of open source technologies to provide you a single API interface and unified management console for the your cloud infrastructure. A short list of technologies we leverage is below:

  • Zabbix - Enterprise level monitoring and health checks for servers.
  • Kibana/Logstash - Log consolidation and forwarding
  • Puppet - For provisioning and bootstrapping
  • Docker - Containers for base images
  • Terraform - Create and compose all the components necessary to run any service or application

and more.
